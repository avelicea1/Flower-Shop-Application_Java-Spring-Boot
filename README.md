# Flower Shop Application - Java Spring Boot

## Description
This project is a **Flower Shop Application** built using **Java Spring Boot**, **Spring Security**, **Thymeleaf**, **Hibernate**, and **MySQL**.  
It implements **user registration and login functionality**, secure authentication, and a simple interface for managing flower shop operations.

The application follows modern web development practices with a focus on **security**, **database integration**, and **template-based UI rendering**.

## Features
- User registration and login with Spring Security
- Secure password storage
- Dynamic web pages using Thymeleaf templates
- Database management with Hibernate and MySQL
- Role-based access control (optional for admin/user functionality)

## Technologies Used
- Java 17+
- Spring Boot
- Spring Security
- Thymeleaf
- Hibernate (JPA)
- MySQL
- Maven (Project Management)
